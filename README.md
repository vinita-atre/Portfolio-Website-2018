# [Vinita Atre's design portolio](http://atrevinita.com)

I used the [Stylish Portfolio's ](http://startbootstrap.com/template-overviews/stylish-portfolio/) one page portfolio theme for [Bootstrap](http://getbootstrap.com/) created by [Start Bootstrap](http://startbootstrap.com/). The theme features multiple content sections with an off canvas navigation menu.
